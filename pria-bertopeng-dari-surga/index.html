Pria Bertopeng dari Surga

Pria Bertopeng dari Surga bercerita tentang negara air dan negara api. Negara api adalah negara paling di benci oleh negara-negara lain namun negara air selalu membantunya, saat negara api telah bangkit dan lebih hebat negara api malah menyerang negara air dan hal tersebut telah diramal kan di sebuah kitab kuno. Di kitab tersebut tertulis bahwa negara api hanya bisa dikalahkan oleh pria bertopeng dari surga. Dikerahkanlah semua orang untuk mencarinya.
